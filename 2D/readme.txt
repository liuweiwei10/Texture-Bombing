Usage:
    compile: make
    run: ./viewer -f triangles.frag


Notes:
“1.frag” - only sample the current cell
“2.frag” - sample adjacent cells and set priority
“3.frag” - use color from radomTexture
“4.frag” - sample 3 stars per cell
“5.frag” - procedural images
“6.frag” - random selection from 5 images
